# 📊 Superstore Sales Analysis Dashboard

## 📌 Project Overview
This project focuses on analyzing Superstore sales data to gain insights into sales performance, profitability, and customer segments. An interactive dashboard was built using Power BI to visualize key metrics and support data-driven decision-making.

---

## 🎯 Objectives
- Analyze overall sales, profit, and quantity performance  
- Identify top-performing categories and sub-categories  
- Understand regional and segment-wise sales distribution  
- Track monthly trends for sales and profit  
- Evaluate shipping mode preferences  

---

## 📊 Dataset Description
The dataset includes the following key fields:

- Order ID  
- Order Date  
- Ship Mode  
- Segment (Consumer, Corporate, Home Office)  
- Region  
- State  
- Category & Sub-Category  
- Sales  
- Profit  
- Quantity  

---

## 🛠️ Tools & Technologies Used
- Power BI (Dashboard & Visualization)  
- DAX (Measures & Calculations)  
- Power Query (Data Cleaning & Transformation)  

---

## 🔍 Key Analysis Performed
- Created KPI cards for **Total Sales, Total Profit, and Total Quantity**  
- Developed **monthly sales and profit trend analysis** using line charts  
- Performed **category and sub-category analysis** to identify best-selling products  
- Analyzed **segment-wise and region-wise sales distribution**  
- Built **state-wise sales map visualization**  
- Evaluated **shipping mode performance**  

---

## 📈 Key Insights
- The **Consumer segment** contributes the highest sales  
- Certain regions dominate overall revenue generation  
- A few sub-categories drive the majority of profit  
- Sales and profit show **seasonal trends across months**  
- Standard Class is the most commonly used shipping mode  

---

## 📊 Dashboard Features
- Interactive filters (Region selection)  
- KPI cards for quick performance overview  
- Line charts for time-based trends  
- Bar charts for category and segment comparison  
- Map visualization for geographical insights
